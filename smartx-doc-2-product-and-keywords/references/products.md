# SmartX 产品线

> 产品识别时**只能从以下列表中选取**，禁止自行添加产品名称。

| 产品名称 | 核心功能 |
|---------|---------|
| SMTX OS & SMTX ELF | 超融合软件：分布式块存储、计算虚拟化、网络、数据保护、运维管理 |
| SMTX ZBS | 分布式存储软件：块存储、文件存储、运维管理 |
| CloudTower | 多集群统一管理平台：虚拟机生命周期、存储/网络/硬件管理、监控报警、RESTful API & SDK |
| CloudTower Commander | 多 CloudTower 统一监控与管理 |
| Everoute | 网络与安全：分布式防火墙、VPC、负载均衡、安全流量导流、容器网络 |
| SMTX 备份与容灾 | 虚拟机与块存储卷的数据保护和灾难恢复 |
| SMTX 文件存储（SFS） | 分布式文件存储：非结构化数据（媒体、图像、音频等） |
| 可观测性平台 | 监控、报警、日志、追踪、网络流量可视化 |
| SMTX Kubernetes 服务（SKS） | 企业级 Kubernetes 集群部署与管理 |
| 榫卯 AI 平台 | 大模型基础设施管理：算力调度、推理服务、模型管理 |

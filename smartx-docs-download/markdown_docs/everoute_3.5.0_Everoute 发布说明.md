## 版本更新说明

# 版本更新说明

## 新增

### 网络安全导流

支持网络安全导流功能，可将虚拟分布式交换机中的流量导流至特定安全服务设备。

在 SmartX 集群中使用深度安全防护功能时，可以通过网络安全导流将特定虚拟网卡的流量导流至安全防护虚拟机，从而为该虚拟网卡上的流量提供深度包检测能力。

### 分布式防火墙

当关联的虚拟分布式交换机开启 IGMP/MLD 侦听功能时，在分布式防火墙中支持默认放通 IGMP/MLD 报文，无需手动配置相关安全策略。

### 通用

适配 CloudTower 高可用功能，支持复用 CloudTower 仲裁节点作为 Everoute Controller 的仲裁节点。

## 改进

### 分布式防火墙

- 自定义安全策略的名称新增支持中文。
- 自定义安全策略的描述信息支持独立展示，并支持搜索与高级筛选。
- 在配置安全策略端口号时，新增格式校验，并支持将空格自动转换为英文逗号分隔。
- 取消对安全组描述信息的字符限制。
- 优化自定义安全策略中对称策略的配置方式及描述信息，明确此配置将对入流量白名单中的源端虚拟机配置允许出站的流量规则，以及对出流量白名单中的目标端虚拟机配置允许入站的流量规则。

### 网络负载均衡器

对虚拟服务的会话保持功能新增**服务器故障处理**特性，默认策略为**立即切换至新服务器**，支持通过命令行手动调整为**继续会话保持**。

## 版本配套说明

# 版本配套说明

## 与 SMTX OS 版本配套说明

- **部署要求**：本版本 Everoute 支持部署在 5.0.1 及以上 5.0.x 版本、5.1.x 版本、6.x.x 版本的 SMTX OS (ELF) 集群上。
- **使用功能的要求**：

  - 本版本 Everoute 的分布式防火墙支持关联 5.0.1 及以上 5.0.x 版本、5.1.x 版本、6.x.x 版本的 SMTX OS (ELF) 集群。
  - 本版本 Everoute 的网络负载均衡器支持关联 5.0.1 及以上 5.0.x 版本、5.1.x 版本、6.x.x 版本的 SMTX OS (ELF) 集群的虚拟机网络。
  - 本版本 Everoute 的虚拟专有云网络支持关联 6.0.0 及以上 6.x.x 版本的 SMTX OS (ELF) 集群，推荐使用 6.1.1 及以上的 6.x.x 版本。如需使用 VPC 边缘网关组功能，建议主、备边缘网关所在的集群保持相同版本。

## 与 SMTX ELF 版本配套说明

- **部署要求**：本版本 Everoute 支持部署在 6.0.0 及以上 6.x.x 版本的 SMTX ELF 集群上。
- **使用功能的要求**：

  - 本版本 Everoute 的分布式防火墙支持关联 6.0.0 及以上 6.x.x 版本的 SMTX ELF 集群。
  - 本版本 Everoute 的网络负载均衡器支持关联 6.0.0 及以上 6.x.x 版本的 SMTX ELF 集群的虚拟机网络。
  - 本版本 Everoute 的虚拟专有云网络支持关联 6.0.0 及以上 6.x.x 版本的 SMTX ELF 集群，推荐使用 6.1.1 及以上的 6.x.x 版本。如需使用 VPC 边缘网关组功能，建议主、备边缘网关所在的集群保持相同版本。

## 与 CloudTower 版本配套说明

本版本 Everoute 只支持使用 CloudTower 4.8.0 及以上 4.x.x 版本进行管理。

## 与可观测性平台版本配套说明

本版本 Everoute 支持配套可观测性平台 1.2.0 及以上 1.x.x 版本，使用网络流量可视化功能。

如需使用安全策略日志及策略计数功能，需配套可观测性 1.3.0 及以上 1.x.x 版本。建议在可观测性平台中将 Everoute 服务关联至一个可观测性服务，从而提供 Everoute 服务相关报警，并缩短策略日志和命中数的数据流量数据的获取和问题排查路径。

## 与 SMTX Kubernetes 服务版本配套说明

本版本 Everoute 支持配套 SKS 1.2.0 及以上 1.x.x 版本使用。

## 版本使用限制

# 版本使用限制

## 分布式防火墙

- 同一个集群只能关联一个 Everoute 服务的分布式防火墙。
- 一个 Everoute 服务的分布式防火墙最多可关联 512 个主机。
- 由于受到系统资源限制，建议在单条自定义安全策略中，作为策略对象和出、入流量名单的虚拟机总数不超过 100 个，开放的出、入流量协议端口不超过 10 个，避免端口范围过大（例如 1 ~ 65535）。

## 网络负载均衡器

- 同一个虚拟机网络只能关联一个 Everoute 服务的负载均衡虚拟机。
- 虚拟服务资源的数量受到 Everoute Controller 配置类型的限制，详见《Everoute 安装与升级指南》的[规划 Everoute Controller](/everoute/3.5.0/er_installation_guide/installation_guide_07) 章节。

## 虚拟专有云网络

- 同一个集群只能关联一个 Everoute 服务的虚拟专有云网络。
- 主机、VPC、路由表等资源的数量受到 Everoute Controller 配置类型的限制，详见《Everoute 安装与升级指南》的[规划 Everoute Controller](/everoute/3.5.0/er_installation_guide/installation_guide_07) 章节。
- 网关资源的数量受到边缘网关虚拟机配置类型的限制，详见《Everoute 虚拟专有云网络管理指南》的[创建边缘网关](/everoute/3.5.0/er_administration_guide_vpc/administration_guide_vpc_50)章节。
- 在虚拟机内部运行容器应用时，若虚拟机使用 VPC 网卡，需要注意以下限制：

  - 容器仅支持 **host** 网络模式或 **overlay** 网络模式，不支持 **bridge** 及其他网络模式。
  - 当使用 **overlay** 网络模式时，容器仅能在 VPC 内部通信，不支持配置浮动 IP 等其他网络功能。

## 网络安全导流

- 同一个集群只能关联一个 Everoute 服务的网络安全导流。
- 每个被关联的集群中只能关联一个虚拟分布式交换机。

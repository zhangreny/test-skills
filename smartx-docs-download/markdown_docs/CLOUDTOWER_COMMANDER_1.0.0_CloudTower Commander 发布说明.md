## 版本特性说明

# 版本特性说明

- 支持汇总展示多个 CloudTower 及特定 CloudTower 管理的资源情况。
- 支持汇总展示多个 CloudTower 管理的集群及系统服务的报警情况。
- 支持筛选资源信息及报警信息。
- 支持将资源信息和报警信息导出为报表。

---

## 版本配套说明

# 版本配套说明

## 与服务器 CPU 服务架构配套说明

当前版本支持运行在 Intel x86\_64 和 Hygon x86\_64 架构的虚拟化平台中。

## 与 SMTX OS 版本配套说明

当前版本支持部署在 5.0.5 及以上版本的 SMTX OS（ELF）集群中。

## 与 CloudTower 版本配套说明

CloudTower Commander 通过关联多个 CloudTower 以对其进行统一管理，CloudTower 须为 3.4.2 及以上版本。

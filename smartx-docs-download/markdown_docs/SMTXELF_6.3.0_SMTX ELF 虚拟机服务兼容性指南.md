# SMTXELF/6.3.0/SMTX ELF 虚拟机服务兼容性指南
## 关于本文档

# 关于本文档

在部署 SMTX ELF 时，软件可直接安装在 Intel x86\_64、AMD x86\_64、Hygon x86\_64 和鲲鹏 AArch64 架构的物理服务器上。本文档列举了 SMTX ELF 基于以上不同类型的物理服务器部署后所支持的虚拟机服务功能，与其客户机的操作系统及版本、系统盘总线和固件类型的兼容性说明，以及 SMTX 虚拟机工具和 Cloud-init/Cloudbase-init 的各项功能与客户机的操作系统及版本的配套支持关系。

单击[下载《SMTX ELF 6.3.0 虚拟机服务兼容性指南》](https://cdn.smartx.com/internal-docs/assets/d9c52888/SMTX ELF 6.3.0 虚拟机服务兼容性指南.xlsx)，可将文档下载至本地电脑查看。

# SMTXOS/6.3.0/SMTX OS 虚拟机服务兼容性指南
## 关于本文档

# 关于本文档

当在 ELF 平台部署 SMTX OS 时，软件可直接安装在 Intel x86\_64、Hygon x86\_64 和鲲鹏 AArch64 架构的物理服务器上。本文档列举了 SMTX OS 基于以上不同类型的物理服务器部署后所支持的虚拟机服务功能，与其客户机的操作系统及版本、系统盘总线和固件类型的兼容性说明。

单击[下载《SMTX OS 虚拟机服务兼容性指南》](https://cdn.smartx.com/docs-admin/uploads-new/SMTX%20OS%20虚拟机服务兼容性指南.xlsx)，可将文档下载至本地电脑查看。

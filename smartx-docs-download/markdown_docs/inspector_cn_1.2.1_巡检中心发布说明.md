## 版本更新说明

# 版本更新说明

## 新增

- 支持英文界面。
- 支持以表格形式导出巡检集群 6个月内的内存分配率信息。
- 新增环境检查项：SMTX ZBS 集群的 Boost 模式启用情况。

## 改进

- 优化了巡检中心对集群连通性探测的端口配置，将默认端口由 `8500` 调整为 `443`。
- 优化了 CloudTower 对集群节点 `443` 和 `10000` 端口访问异常的提示信息，显著提升了运维排查效率。
- 优化了巡检中心在集群中部署 agent 时的超时时间设置。

## 修复问题

- 修复了导出的巡检报告的正文字体不一致的问题。
- 修复了对 5.1.4-P2 和 5.1.5-P1 版本的 SMTX OS 集群巡检时，系统误报**日志回滚配置目录检查**巡检项存在风险的问题。

## 版本配套关系

# 版本配套关系

## 与 CloudTower 版本配套说明

巡检中心是部署在 CloudTower 虚拟机中的一项服务，此版本巡检中心服务由 4.8.0 及以上版本的 CloudTower 提供。 使用巡检中心服务前，请确认 SMTX OS 集群、SMTX ELF 集群和 SMTX ZBS 集群的主机的管理网络的 443 和 10000 端口已开放。

> **说明**：
>
> 对于部署在 CloudTower 4.7.0 ～ 4.7.1 中的巡检服务，升级至 1.2.1 版本之后，不支持英文界面。

## 与集群版本配套说明

| 集群类型 | 软件版本 |
| --- | --- |
| SMTX OS（ELF）集群 | 5.0.5 及以上的 5.0.x、5.1.x 和 6.x.x |
| SMTX OS（VMware ESXi）集群 | 5.0.6-HP2 及以上的 5.0.x、5.1.x 和 6.x.x |
| SMTX ELF 集群 | 6.0.0 及以上的 6.x.x |
| SMTX ZBS 集群 | 5.5.0 及以上的 5.x.x |

## 与 VMware vSphere 版本配套说明

SMTX OS（VMware ESXi）集群使用巡检中心服务需要配套如下 VMware vSphere 版本：

- ESXi：

  - 6.5 P06 及后续版本
  - 6.7 P03 及后续版本
  - 7.0b、7.0 U1、7.0 U2、7.0 U3f、7.0 U3g
  - 8.0 U1a、8.0 U3
- vCenter：

  - 6.7 及更高的 6.x 版本
  - 7.0 版本

## 版本使用限制

# 版本使用限制

巡检中心 1.2.1 以下版本生成的报告仅支持中文。切换至英文界面后，您将无法查看或导出其英文版本。

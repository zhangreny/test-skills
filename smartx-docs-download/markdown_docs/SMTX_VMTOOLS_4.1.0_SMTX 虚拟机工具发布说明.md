## 版本更新说明

# 版本更新说明

## 新增

- 支持为虚拟机安装 DS 驱动以启用防病毒服务。
- 支持 Windows Server 2025 操作系统。

## 优化

升级了虚拟机工具携带的 virtio-win 驱动版本。

## 修复问题

- 修复了虚拟机移除磁盘后残留挂载点可能导致创建文件系统一致性快照失败的问题。
- 修复了虚拟机移除磁盘后残留挂载点可能导致文件系统分区使用率信息获取失败的问题。
- 修复了操作系统为 UOS 20 1070a 的虚拟机可能因进程耗尽导致虚拟机服务异常的问题。
- 修复了虚拟机工具可能因虚拟机 udev 规则目录不符合预期导致安装失败的问题。
- 修复了虚拟机工具可能因 nmcli 命令卡住导致安装失败的问题。
- 修复了 Windows 虚拟机安装虚拟机工具后设备管理器中可能出现带有警告标识 virtio 设备的问题。

## 版本配套说明

# 版本配套说明

## 与集群版本配套说明

SMTX 虚拟机工具需安装在 ELF 虚拟机中，本版本的虚拟机工具支持以下集群类型及版本：

- SMTX OS 5.1.5 及以上的 5.1.x 版本
- SMTX OS 6.2.0 及以上的 6.2.x 版本
- SMTX OS 6.3.0 及以上的 6.3.x 版本
- SMTX ELF 6.2.0 及以上的 6.2.x 版本
- SMTX ELF 6.3.0 及以上的 6.3.x 版本

## 与虚拟机操作系统版本配套说明

本版本的 SMTX 虚拟机工具在不同客户机操作系统中支持的虚拟机服务功能可能存在差异，具体支持情况请参考[《SMTX 虚拟机工具兼容性列表》。](https://cdn.smartx.com/internal-docs/assets/adf7f501/SMTX 虚拟机工具 4.1.0 兼容性列表.xlsx)

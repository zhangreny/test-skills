## 版本更新说明

# 版本更新说明

## 新增

为 SFS 1.3.0 提供预置的 Grafana Dashboard，以应对常见售后问题排查与性能分析，提升排查效率。

## 改进

- 优化了 notifycenter 对 TLS 加密算法的兼容性，避免因算法不支持导致邮件配置失败。
- 支持在网络流量可视化中展示网络安全导流相关流量。
- 支持监控网络流量可视化中组件的缓存项并估算内存使用情况，在内存不足时自动对可用性进行降级。

## 修复问题

- 修复了 SMTX OS 5.0.x 和 5.1.x 集群因无有效数据块导致监控数据无法迁移至可观测性服务的问题。
- 修复了集群重新关联可观测性服务时，因旧 namespace 残留导致关联失败的问题。
- 修复了预置 Grafana Dashboard 中 `host_cpu_fan_is_ok` 变量无法获取的问题。
- 修复了因可观测性服务报警通知任务延迟，导致触发报警功能异常的问题。
- 修复了网络流量可视化在获取集群数据时连接泄漏，导致 CloudTower 不可用的问题。

## 版本升级注意事项

# 版本升级注意事项

- 如果从 1.2.0 以下版本升级至本版本，请先参照《可观测性平台用户指南》的[规划可观测性服务的配置](/visualization_cn/1.4.3/visualization_user_guide/visualization_user_guide_04#%E8%A7%84%E5%88%92%E5%8F%AF%E8%A7%82%E6%B5%8B%E6%80%A7%E6%9C%8D%E5%8A%A1%E7%9A%84%E9%85%8D%E7%BD%AE)内容对可观测性虚拟机进行扩容。
- 对于 1.3.0 以下版本的可观测性服务，若升级前已启用网络流量可视化功能，请提前联系 SmartX 售后工程师协助更新网络流量可视化许可。

## 版本配套说明

# 版本配套说明

## 与集群版本配套说明

可观测性服务虚拟机部署在集群内，并通过关联集群为其提供可观测性服务，同一可观测性服务可支持关联多种 CPU 架构的集群。具体的版本支持情况如下表所示。

| 集群类型 | 是否启用双活特性 | 版本要求 | |
| --- | --- | --- | --- |
| 关联可观测性 1.4.3 | 部署可观测性 1.4.3 |
| SMTX OS（ELF）集群 | 否 | 5.0.5 及以上的 5.0.x、5.1.x、6.x.x | 5.0.5 及以上的 5.0.x、5.1.x、6.x.x |
| 是 | 6.0.0 及以上的 6.x.x | 5.1.3 及以上的 5.1.x、6.x.x |
| SMTX OS（VMware ESXi）集群 | 否 | 5.0.6 Hotfix Package2 及以上的 5.0.x、5.1.x、6.x.x | 不支持 |
| 是 | 6.0.0 及以上的 6.x.x | 不支持 |
| SMTX ZBS 块存储集群 | 否 | 5.5.0 及以上的 5.x.x | 5.6.0 及以上的 5.x.x |
| 是 | 5.6.0 及以上的 5.x.x | 5.6.0 及以上的 5.x.x |
| SMTX ELF 集群 | 否 | 6.0.0 及以上的 6.x.x | 6.0.0 及以上的 6.x.x |

> **说明**：
>
> - SMTX ELF 目前无双活集群。
> - 强烈建议所有 SMTX OS 6.0.0 及以上的 6.x.x 集群和 SMTX ELF 6.0.0 及以上的 6.x.x 集群均与可观测性服务进行关联，由可观测性服务为集群提供统一的指标采集、存储、处理以及更加完善的监控能力。
>   - 若低版本的 SMTX OS 集群未部署高级监控应用，且欲升级至 SMTX OS 6.0.0 及以上的 6.x.x 版本，请先将集群和可观测性服务关联并执行数据迁移，然后再将集群升级至最新版本，以避免原有的监控数据丢失。
>   - 若集群已部署高级监控应用，请在关联可观测性服务后执行数据迁移，完成可观测性对高级监控的替换。

## 与系统服务版本配套说明

| 系统服务 | 配套说明 |
| --- | --- |
| CloudTower | 仅支持由 CloudTower 4.7.0 及以上的 4.x.x 版本进行管理，且支持关联当前 CloudTower 以为其提供监控报警功能。 |
| Everoute | 网络流量可视化功能支持与 Everoute 1.1.2 及以上的 1.x.x 版本、2.x.x 版本、3.x.x 版本搭配使用。  支持关联 Everoute 3.1.0 及以上的 3.x.x 版本以为其提供监控报警功能。 |
| SMTX 文件存储 | 支持关联 SMTX 文件存储 1.1.0 及以上的 1.x.x 版本以为其提供监控报警功能。 |
| SMTX 备份与容灾 | 支持关联 SMTX 备份与容灾 2.1.0 及以上的 2.x.x 版本以为其提供监控报警功能。 |
| SMTX Kubernetes 服务 | 网络流量可视化功能支持与 SMTX Kubernetes 服务 1.3.2 及以上 1.x.x 版本搭配使用。  支持关联 SMTX Kubernetes 服务 1.4.0 及以上的 1.x.x 版本以为其提供监控报警功能。  支持关联 SMTX Kubernetes 服务 1.5.0 及以上的 1.x.x 版本以为其提供日志功能。 |

## 关于本文档

# 关于本文档

本文档介绍了集群上下线工具 1.0.4 的版本新增功能、修复问题、版本配套说明和产品配套文档。

## 文档更新信息

# 文档更新信息

**2026-02-04：文档随集群上下线工具 1.0.4 正式发布**

首次发布集群上下线工具 1.0.4 发布说明。

---

## 版本更新说明

# 版本更新说明

## 新增

- 支持对 SMTX 文件存储 1.2.0、1.2.1 以及 1.3.0 版本的系统服务虚拟机进行开关管理。
- 支持对 SMTX Kubernetes 服务 1.4.0、1.5.0 以及 1.5.1 版本的系统服务虚拟机进行开关管理。
- 支持对启用了深度安全防护的 SMTX OS 6.3.0 集群进行上线和下线操作。
- 支持对自定义 SSH 端口的 SMTX OS 6.3.0 集群进行上线和下线操作。

## 修复问题

修复了虚拟机自启动与集群上下线工具的开机指令发生冲突，可能导致集群上线失败的问题。

---

## 版本配套说明

# 版本配套说明

集群上下线工具支持同时管理 SMTX 集群和多个产品的上下线操作，但集群版本、CloudTower 版本以及其他产品版本需要与集群上下线工具兼容，建议使用以下推荐的产品版本组合。

> **注意**：
>
> 当集群由 CloudTower 4.8.0 及以上版本管理时，若 CloudTower 已启用高可用功能，则不支持使用集群上下线工具执行上线和下线操作，请联系 SmartX 售后工程师获取集群上下线步骤。

---

# everoute/3.5.0/Everoute 网络安全导流管理指南
## 关于本文档

# 关于本文档

Everoute 是由北京志凌海纳科技股份有限公司（以下简称 “SmartX”）自主研发的网络与安全产品，为 SmartX 超融合和企业云基础设施提供了软件定义的网络和安全系列功能。Everoute 可提供分布式防火墙、网络安全导流、负载均衡、虚拟专有云网络、容器网络等功能，同时支撑虚拟化和容器化应用，从而形成与虚拟机和容器一体化管理的网络和安全解决方案。

本文档介绍了如何使用 Everoute 网络安全导流的功能特性。阅读本文档的对象需了解 SMTX OS 超融合软件、SMTX ELF 虚拟化软件以及 Everoute 服务的相关技术背景知识，并具备丰富的数据中心操作经验。

>
> 由于 Everoute 和 CloudTower 软件对外提供多个版本，且每个版本的功能和服务均有所不同。为了完整描述 Everoute 软件的功能和特性，本⽂档中所有功能均基于 Everoute 3.5.0 和 CloudTower 4.8.0。若您所购买的软件版本不包含文档中描述的部分功能和特性，请忽略并以实际使用的软件功能为准。

## 概述

# 概述

Everoute 网络安全导流（Security traffic steering）可支持在 SmartX 原生虚拟化 ELF 中，将特定虚拟分布式交换机中的出入网络流量导流至特定的安全服务设备，以实现对网络流量的深度检测、入侵防御、防火墙等能力，保证虚拟机中的网络数据安全。

如需在 SmartX 集群中部署深度安全防护功能，可通过 Everoute 网络安全导流将特定虚拟网卡中的流量导流至安全防护虚拟机（Security Virtual Machine，简称 “SVM”），对流量包进行深度检测，并提供 Web 信誉、防火墙、入侵检测及虚拟补丁等功能。

## 启用网络安全导流

# 启用网络安全导流

部署 Everoute 服务时，您可以启用网络安全导流功能；如未在部署阶段启用，也可参考以下操作说明单独启用该功能。

**前提条件**

确认已开启相应的防火墙端口，请参考《Everoute 安装与升级指南》的[规划端口](/everoute/3.5.0/er_installation_guide/installation_guide_08#%E7%BD%91%E7%BB%9C%E5%AE%89%E5%85%A8%E5%AF%BC%E6%B5%81)章节。

**操作步骤**

1. 进入 CloudTower 的**网络与安全**主页面，在左侧导航栏中单击**设置**，选择要启用网络安全导流的 Everoute 服务。
2. 在**服务设置**里单击**基本信息**。
3. 在**网络安全导流**功能右侧单击**启用功能**。
4. 确认启用功能后，单击**启用**。

   启用后，您需要为网络安全导流功能[关联集群](/everoute/3.5.0/er_administration_guide_sts/administration_guide_sts_04)，为对应集群上的虚拟机提供导流能力。

## 查看概览

# 查看概览

进入 CloudTower 的**网络与安全**主页面，您可以在概览页查看当前所有已部署的 Everoute 服务的信息。

每个 Everoute 服务概览卡片包括如下内容：

- **服务名称**：卡片标题即为 Everoute 服务的名称。
- **版本**：展示当前 Everoute 服务的版本。当存在可升级的版本时，单击**可升级**，可进入**服务升级**页面，选择版本完成升级操作。
- **Controller 状态**：展示 Everoute Controller 的数量及当前状态。单击 **Controller 状态**可查看 Everoute Controller 信息。
  > **说明**：
  >
  > - Controller 状态包括正常运行、异常、正在部署、正在卸载。
  > - 绿色表示正常状态的 Controller 个数，红色表示异常状态（包括异常、正在部署、正在卸载）的 Controller 个数，黄色表示 vCPU、内存、存储容量使用率大于 80% 的 Controller 个数。
- **网络安全导流**：若当前 Everoute 服务已开启网络安全导流功能，将展示已关联的集群数量，以及 Everoute 代理异常的数量；否则将隐藏该项。
  - 当 Everoute 代理异常时，可显示异常代理数量，单击**查看详情**可查看具体异常的主机信息。
  - 若当前服务未关联集群，单击**关联集群**，可添加集群关联。
  - 单击**网络安全导流**进入功能设置页面，可管理关联集群。

## 关联集群

# 关联集群

启用网络安全导流后，需要关联指定的 SMTX OS（ELF）集群或 SMTX ELF 集群、以及虚拟分布式交换机，并在虚拟分布式交换机所关联的每个主机的 SVM 上指定一对导流网卡。

完成关联后，若关联集群的虚拟机的虚拟网卡已开启深度包检测功能，则 Everoute 可将相应虚拟网卡的业务流量导入 SVM 进行深度检测，以实现对流量的安全分析与威胁识别。

**关联规则**

- 一个 Everoute 服务可关联多个相同或不同 CPU 架构的集群，但一个集群仅能关联一个 Everoute 服务。
- 一个 Everoute 服务在每个关联集群上仅能关联一个虚拟分布式交换机。

**前提条件**

- 待关联的集群的服务器 CPU 架构为 x86\_64 且未启用 Boost 模式。
- 待关联的集群未关联其他 Everoute 服务。
- 待关联的虚拟分布式交换机的关联主机上已创建 SVM，并且 SVM 状态为`运行中`。
- 待关联的虚拟分布式交换机未关联已启用 RDMA 功能的存储网络。

**注意事项**

关联集群可能对集群网络产生影响，导致网络丢包，建议在业务空闲时或业务下线后谨慎操作。

**操作步骤**

1. 进入 CloudTower 的**网络与安全**主页面，在左侧导航栏中单击**设置**，选择待关联集群的 Everoute 服务。
2. 在**服务设置**中单击**网络安全导流**，在右侧页面单击**添加关联集群**。
3. 选择待关联的**集群**和**虚拟分布式交换机**。
4. 为虚拟分布式交换机已关联的每个主机指定一对导流网卡。

   若主机**不配置导流网卡**，则该主机上的虚拟机将无法使用深度包检测功能，建议为所有主机统一配置导流网卡。
5. 单击**关联**。

   完成关联操作后，可查看已关联的集群、虚拟分布式交换机以及虚拟分布式交换机关联主机对应的导流网卡信息。

## 编辑或解除关联集群

# 编辑或解除关联集群

您可以按需更换关联的虚拟分布式交换机，取消配置导流网卡，或者解除 Everoute 服务和集群的关联。

此外，在集群扩容或虚拟分布式交换机添加关联主机后，也需要根据本节操作说明为新增主机配置导流网卡。

**前提条件**

- 若需更换虚拟分布式交换机，请确保原交换机关联主机上的所有虚拟机的所有虚拟网卡已关闭深度包检测功能。
- 若某主机需设置为**不配置导流网卡**，请确保该主机上的所有虚拟机的所有虚拟网卡已关闭深度包检测功能。
- 若需解除关联集群，请确保集群中的所有虚拟机的所有虚拟网卡已关闭深度包检测功能。

**风险提示**

取消关联集群可能对集群网络产生影响，导致网络丢包，建议在业务空闲时或业务下线后谨慎操作。

**操作步骤**

1. 进入 CloudTower 的**网络与安全**主页面，在左侧导航栏中单击**设置**，选择待编辑或解决关联集群的 Everoute 服务。
2. 在**服务设置**中单击**网络安全导流**。
3. 在某一关联集群右侧按需单击**编辑**或**解除关联**，在弹出的对话框中进行相应操作。

## 管理权限

# 管理权限

Everoute 通过 CloudTower 进行运维管理。

CloudTower 提供两种类型的角色：默认角色和自定义角色，并对不同角色赋予不同的操作权限。其中默认角色的权限不可更改，自定义角色可根据使用需求自行创建。

Everoute 支持在创建自定义角色时，对角色按照下表可分配权限项的细分粒度赋予其操作权限。详细步骤可参见《CloudTower 使用指南》的**管理用户与权限** > **角色管理** > **创建自定义角色**小节。

- **Everoute 服务及许可权限说明**

  | 类别 | 可分配权限项 | 对应的操作权限 |
  | --- | --- | --- |
  | 软件许可 | 许可管理 | 更新许可 |
  | Everoute 安装包 | 安装包管理 | - 上传安装包 - 删除安装包 |
  | Everoute 服务 | 部署 Everoute 服务 | 部署 Everoute 服务 |
  | 卸载 Everoute 服务 | 卸载 Everoute 服务 |
  | 编辑 Everoute 服务 | - 提高 Everoute Controller 数量 - 提高 Everoute Controller 配置类型 - 编辑 Everoute Controller 节点信息 - 节点替换 |
  | 升级 Everoute 服务 | 升级 Everoute 服务 |
- **网络安全导流功能权限说明**

  | 类别 | 可分配权限项 | 对应的操作权限 |
  | --- | --- | --- |
  | 网络安全导流 | 网络安全导流功能管理 | - 启用网络安全导流 - 卸载网络安全导流 |
  | 集群关联管理 | - 添加关联集群 - 编辑关联集群 - 删除关联集群 |

## 卸载网络安全导流

# 卸载网络安全导流

当您不再需要某个 Everoute 服务中的网络安全导流功能，想要卸载该功能时，请在确认满足卸载的要求后，参照操作步骤进行卸载。

**卸载要求**

Everoute 服务的网络安全导流功能中没有关联的集群。

**操作步骤**

1. 登录 CloudTower，进入**网络与安全**主页面，选择**设置**，选择待卸载功能的 Everoute 服务。
2. 在**服务设置**中单击**基本信息**，进入基本信息界面。
3. 在待卸载功能名称右侧单击**卸载功能**。
4. 确认卸载功能后，单击**卸载**。

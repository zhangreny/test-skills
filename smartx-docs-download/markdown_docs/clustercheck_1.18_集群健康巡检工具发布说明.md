## 版本更新说明

# 版本更新说明

## 改进

- 适配 SMTX OS 5.1.4 IO 重路由逻辑，避免新逻辑产生检查误报。
- 调整巡检项检查日志占用量的阈值，由 8 GiB 增加至 20 GiB。
- 优化巡检报告表格的字段展示和文字样式。
- 在巡检报告的集群环境概览中增加集群物理核总数信息。

## 修复问题

- 修复了当禁用低于 SMTX OS 5.1.0 版本集群的 `ESXi shell` 和 `ssh` 服务时，无法获取 IO 路由状态的问题。
- 修复了在环境信息的事件检查中，在没有获取到检查数据时也会展示为`存在告警`的问题。

## 版本已知问题

# 版本已知问题

- 巡检报告的目录部分⻚码显示错误，需手动更新目录。
- 低于 5.0.5 版本的 SMTX OS 集群若安装了本版本巡检工具，在集群升级至 5.0.5 及更高版本后，需重新部署巡检工具，否则 `cluster_server_data.json` 文件无法获取 product info 和 disk info 数据。
